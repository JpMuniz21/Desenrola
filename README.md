## 🛠️ Tecnologias & Segurança (RF06)
- **Front-end:** React.js
- **Back-end:** Node.js + Express
- **Database:** Supabase
- **Segurança:** BCrypt 
- **Ambiente:** Variáveis protegidas via `.env`

## 🚀 Como Rodar o Projeto
### 1. Back-end
```bash
cd back
npm install
npm install amqplib
# Crie seu .env baseado no .env.example
node index.js
```
### 2. Front-end
```bash
cd front
npm install
npm run dev
```
